## Background

Smart Archie is an AI-driven assistant that converts conversational input into cloud architecture diagrams to accelerate migration planning.

## Approach

- Collected requirements via interviews and surveys
- Built a prompt-engineered workflow using LLMs and templating
- Integrated with cloud provider icons and export to PNG

## Results

- Reduced initial architecture drafting time by ~60%
- Helped stakeholders understand cost and complexity trade-offs
